0 Создайте папку для работы с Jinx> например "work"
1. Распаковать архив jinx-2.4.zip в созданную папку work
2. Скачайте архив с gif-картинками, использующимися в эффектах из https://drive.google.com/file/d/1kcc4bdjTrMEvEoL1LI-N9i0ZEnaoGTZM/view?usp=sharing   
   Создайте папку 'pics' в папку work (рядом с папкой 'jinx') распакуйте туда gif-ки из скачанного архива pics.7z
3. В папку work поместить *.jnx файлы с эффектами
4. Запустить jinx.exe из папки jinx
5. Открыть выбранный файл с эффектами *.jnx

Эффекты используют скрипт "Image Viewer" который проигрывает анимацию gif-ок, 
размещенных в папке pics, поэтому такая структура папок - обязательно, иначе
Image Viewer не найдет картинку на основе которой создан эффект.
Если gif-ка не произрывается - нажмите кнопку 'Edit' у скрипта с Image Viewer
и найдите по имени файла картинку, которую тот использует.
